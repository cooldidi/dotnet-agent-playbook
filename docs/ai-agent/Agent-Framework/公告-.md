# Agent Framework 系列示例升级公告

花了一周时间，我们将之前 Agent Framework 系列文章的所有配套示例进行了全面升级，涉及 30 多个案例。

由于早期案例发布时 Agent Framework 正式版尚未发布，版本更新过程中有一些破坏性的更新，比如有些方法在正式版本发布之后就没有了等问题。因此我们将示例落到一个稳定版本之后，对配套示例进行了重新整理和升级，并完成了兼容性验证。

同时，在兼容 `AzureOpenAIClient` 的基础上，我们增加了对 `AIProjectClient` 的支持，方便大家在本地进行 Agent Framework 的开发和调试。

## 客户端介绍

- **`AzureOpenAIClient`**：Azure OpenAI 服务客户端，用于连接 Azure OpenAI 服务。它封装了 OpenAI API，提供了更便捷的接口调用方式。
- **`AIProjectClient`**：Microsoft Foundry Project 服务客户端，用于连接 Foundry Project，并获取各种模型的推理能力等功能。

`AIProjectClient` 不但继承了 `AzureOpenAIClient` 的所有功能，还增加了更多能力。

## `AzureOpenAIClient` 关联包升级
